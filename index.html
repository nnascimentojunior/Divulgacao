Projeto 1 Campus Acessível (IA e Inclusão)

Use IA para tornar o campus mais acessível para pessoas com deficiência visual

Google Teachable Machine Visão Computacional Aplicação Web Guia Virtual

O Desafio

Como podemos usar a tecnologia para tornar nosso campus mais inclusivo para pessoas com deficiência visual?

A Solução

Você vai ajudar a desenvolver uma aplicação web usando Inteligência Artificial e Visão Computacional. O sistema funcionará como um “guia virtual”: o celular reconhecerá o ambiente (biblioteca, laboratório, etc.) e falará o nome do local para o usuário.

Por que se candidatar?

  • Aprenda na prática sobre Machine Learning e IA sem precisar de matemática complexa.
  • Desenvolva uma solução com impacto social direto e visível na vida das pessoas.
  • Trabalhe com tecnologias web modernas e boas práticas de desenvolvimento.
Projeto 2 Game Campi Conectados (Jogos e Criatividade)

Crie um jogo 2D para receber calouros de forma divertida e gamificada

GDevelop RPG / Aventura 2D Onboarding de Calouros Gamificação

O Desafio

Como podemos tornar a chegada dos novos alunos (calouros) mais divertida e menos confusa?

A Solução

Vamos criar um jogo digital 2D (estilo RPG/Aventura) usando a engine GDevelop. O jogador explorará um campus virtual, cumprirá missões e aprenderá sobre a história, regras e locais importantes do instituto de forma gamificada.

Por que se candidatar?

  • Aprenda o ciclo completo de desenvolvimento de jogos: roteiro, design de fases, arte e programação visual.
  • Use sua criatividade para resolver um problema real de comunicação com os calouros.
  • Crie um produto que poderá ser jogado por centenas de estudantes todos os anos.

Quem estamos procurando?

Não exigimos experiência prévia com IA ou criação de jogos. O mais importante é ter vontade de aprender, responsabilidade e disposição para colaborar em equipe.
